# StudyFlow

StudyFlow is a full-stack study planner that helps students manage courses, track tasks, generate weekly study plans, and automatically create study blocks before deadlines.

## Features

- Create and manage courses
- Add tasks with deadlines, estimated hours, and priority
- Mark tasks as todo or done
- Generate a weekly plan based on course study-hour budgets
- Generate schedule blocks before deadlines
- View dashboard statistics for courses, tasks, planned hours, and scheduled blocks

## Tech Stack

- Backend: Node.js, Express
- Database: SQLite
- Frontend: HTML, CSS, JavaScript
- Tools: Git, GitHub
